# `coderclaw pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
coderclaw pairing list whatsapp
coderclaw pairing approve whatsapp <code> --notify
```
